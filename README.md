
🌌 Sovereign V14 Singularity
Sovereign V14 Singularity is an ultra-high-performance machine learning framework designed for automated, calibrated, and interpretable classification. It bridges the gap between raw data and production-ready "Neural Synthesis" using a refined XGBoost architecture.

🚀 Key Features
Bayesian Optimization: Automated hyperparameter tuning via BayesSearchCV.

Dual-Stage Calibration: Implements Isotonic and Sigmoid scaling to ensure probability reliability.

Smart Preprocessing: Integrated ColumnTransformer with Robust Scaling and Quantile Transformations.

Hardware Acceleration: Native CUDA support for GPU-accelerated training.

Explainable AI (XAI): Built-in SHAP integration for deep feature insight.

Production Ready: Fully compatible with the scikit-learn API (fit/predict).
